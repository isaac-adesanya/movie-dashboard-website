# Movie Dashboard Website

A front-end movie dashboard that displays real-time movie data using The Movie Database (TMDB) API.

## Technologies Used
- HTML
- CSS
- JavaScript
- jQuery
- TMDB API

## Features
- Displays highest grossing movies from TMDB
- Fetches real-time movie data from the TMDB API
- Shows key information such as title, poster, rating, reviews, and overview
- Dynamic content rendering using JavaScript and jQuery
- Responsive layout with clean styling

## What I Learned
- Working with third-party REST APIs in a front-end application
- Handling asynchronous requests using JavaScript (fetch / AJAX)
- Manipulating the DOM with jQuery
- Structuring a small front-end project for readability
